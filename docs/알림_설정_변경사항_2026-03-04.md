# 알림 설정 변경사항 (2026-03-04)

## 1) 알림 설정 화면
- 알림 설정 화면 UI를 피그마 기준으로 정리했습니다.
- `기기 알림이 꺼져 있어요!` 배너 탭 시 즉시 이동이 아닌 확인 팝업을 먼저 노출합니다.
- 팝업에서 `설정하기` 선택 시 기기 알림 권한 설정 화면으로 이동하고, `취소` 시 현재 화면으로 복귀합니다.

## 2) 오늘의 질문 알림
- 오늘의 질문 토글 ON + 알림 권한 허용 시 `오늘의 질문 알림 시간` 행이 노출됩니다.
- 시간 선택은 바텀시트 휠 피커 방식으로 변경했습니다.
- 기본 시간은 `오후 3:00`입니다.
- 사용자가 선택한 시간/토글 상태를 로컬에 저장하고, 재진입 시 복원합니다.
- 설정 시간 기준으로 매일 로컬 푸시 알림을 스케줄링합니다.

## 3) 버킷리스트 디데이 알림
- `버킷리스트 디데이` 토글 ON 시 디데이 설정 바텀시트를 노출합니다.
- 기본값은 `7일 전`이며 로컬 저장/복원됩니다.
- 버킷리스트 완료 예정일(dueDate) 기준으로 디데이 알림을 재계산해 스케줄링합니다.
- 버킷리스트 추가/수정/완료/삭제 시 알림 스케줄을 동기화합니다.
- 알림 문구:
  - 제목: `{버킷리스트 제목} 완료 D-N일 전이에요!`
  - 본문: `실천하기 위한 계획을 세워볼까요?`

## 4) 앱/플랫폼 설정
- 앱 표시 이름을 `Daily Question`으로 반영했습니다.
- Android 알림 권한(`POST_NOTIFICATIONS`) 및 관련 설정을 반영했습니다.
- `flutter_local_notifications`, `flutter_timezone`, `timezone` 의존성을 추가했습니다.
- Android desugaring 설정을 추가해 알림 라이브러리 빌드 이슈를 해결했습니다.
