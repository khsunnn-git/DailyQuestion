# 개발 메모 (한글)

## 2026-03-04 설정/닉네임 반영
- 더보기 > 설정 메인 화면을 피그마 기준으로 보정
  - 프로필 닉네임을 로컬 저장값으로 노출 (`{닉네임}`)
  - 프로필 이미지를 회원가입 닉네임 물고기 에셋으로 통일
- 프로필 통계 로더 추가
  - 질문 연속 기록일: `TodayQuestionStore` 기반 조회
  - 버킷리스트 개수: 로컬 DB(Isar) 카운트 조회
  - 버킷리스트 개수 표기 규칙 반영: 99개 이상이면 `99+`
- 설정 화면 연필 아이콘 동작 연결
  - 탭 시 닉네임 수정 화면으로 이동
  - 수정 후 설정 화면으로 복귀 시 닉네임/통계 영역 즉시 갱신
- 닉네임 설정 화면 편집 모드 추가
  - 설정 진입 시 문구: `나를 표현할 새로운 닉네임을 적어주세요!`
  - 버튼 라벨: `완료`
  - 저장 완료 시 완료화면으로 가지 않고 설정 화면으로 복귀
- 버킷리스트 없음 화면 피그마 기준 보정
  - 배경 톤/중앙 레이아웃/말풍선/추가 버튼 배치 반영
  - 물고기 이미지 에셋 교체 및 `둥둥` 상하 애니메이션 추가
- 오늘의 기록(타인 기록) 화면 복귀 안정화
  - 상단 뒤로가기 동작을 `pop 가능 시 pop, 불가 시 홈 이동`으로 보완
